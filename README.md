# Hackathon_ETS
3rd place winner of the AMC Hackathon at ETS

# Description
This project was conceived from +he theme "Environmental techonology for a better future". This is a flutter based mobile app that can scan product barcodes and give you a rating out of 100 for the environmental impact it has from its creation to once disposed.

# Tech Stack 
Flutter (Dart), Firebase (Auth + Database), etc
